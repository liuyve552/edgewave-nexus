一句话简介：
把「边缘 RPC 加速」「链上数据可视化」「AI 洞察报告」三件事揉进一个评委友好的演示站：既能 1 分钟验收，也能零配置稳定运行。

评委 1 分钟验收路径：
1) 打开 /demo（Edge Sniper 对比演示）
   - 点击“刷新 / 测量交易确认”，观察左侧基线与右侧 EdgeWave 的延迟差异与最快路径。
   - 基线侧包含人为延迟，用于让对比更“秒懂”。
2) 回到首页 /（3D DeFi 星系）
   - 拖动旋转视角；悬停星球查看 TVL/成交量；点击固定右侧详情卡。
3) 同页下方（AI 代理层：链上洞察）
   - 直接提问（示例见下），获得结构化报告；默认无需任何 Key/外部服务即可复现。

示例提问（任选其一）：
- 给我当前 DeFi 信号的总览。
- 目前 TVL 领先的是哪个协议？
- 哪些协议处于降级状态？可能原因是什么？
- 对比 Uniswap 和 Compound 的动量。

技术实现与亮点（对应评审维度）：
【技术探索】
- ESA Pages 静态部署：采用 Next.js 静态导出（out/），保证评委侧打开即可运行、无需后端。
- 边缘加速思路可量化：/demo 用 Performance API + iframe 消息桥记录 FCP 与 RPC/Receipt 延迟，并展示“最快成功路径”。
- Web3 链上聚合：30 秒节奏读取 Uniswap V3 / Aave / Compound 的代表性只读信号，聚合成结构化指标并用于 3D 可视化。
- ESA 边缘生态补齐：通过 `esa.jsonc` + `edge/index.js` 将边缘 API 与 Pages 同域部署（/edge/rpc、/edge/defi、/edge/ai），减少评委验收配置成本。
- EdgeKV 多级缓存：/edge/defi 采用 L1 内存 → L2 ESA EdgeKV（TTL=30s）→ L3 实时链上读取；响应头 `x-edgewave-defi-cache` 标识命中层级。
- AI 可靠演示策略：默认启用“确定性洞察生成器”（本地可复现）；如启用边缘 AI 端点将自动切换为边缘流式模式（可选能力）。

【应用价值】
- 可复用的“边缘加速 + 可观测”方案：适用于链上看板、RPC 可靠性优化、Web3 前端性能对比验证。
- 不依赖外部服务也能跑：评委侧不需要 Key/额度/模型服务，避免演示翻车；有条件时再接入边缘函数增强说服力。

【创意卓越】
- 用 3D DeFi 星系把抽象指标变成“电影级”交互：悬停注解 + 点击固定详情 + 动态能量线，直观呈现链上状态变化。
- 同站点给出“对比证明”（/demo）与“可视化叙事”（首页 + AI），减少理解成本。
